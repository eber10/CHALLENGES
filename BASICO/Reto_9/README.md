# 💪😎RETO 9 DE PROGRAMACIÓN 🐧

## 🎉Descripción
---
**Enunciado**
 "Comprueba si una palabra contiene una letra concreta"
**Entrada**
- Introducir una palabra/frase.
- Introducir letra.
**Salida**
- Mostrar si está o no la letra correspondiente.
**Ejemplo**
## Entrada
- > Hola esto es una frase
- > N
## Salida
- > N si está.
